# (Yet another) CHIP-8 Interpreter written in C++

## Usage

The code can be compiled via **CMake**.

```bash
cmake -B build
cmake --build build
./build/chip8 <ROM_PATH>
```

Example roms can be found in the **roms** directory.

Uses **Sokol** for rendering and audio.
